## Base de dados
Dataset de vendas de loja disponível no kaggle.

Link: https://www.kaggle.com/datasets/tanayatipre/store-sales-forecasting-dataset

## Objetivo da Análise
Analisar dados de vendas de uma loja para identificar padrões de consumo, produtos mais vendidos e comportamento das vendas ao longo do tempo.

## Planejamento das Tarefas
- Escolha e análise da base de dados
- Limpeza e transformação dos dados
- Implementação do processo de etl
- Armazenamento dos dados em banco de dados
- Criação de um dashboard interativo

## Ideia Inicial do Dashboard
O dashboard deverá apresentar visualizações como:
- Faturamento total
- Vendas por categoria
- Vendas por região
- Evolução das vendas ao longo do tempo
